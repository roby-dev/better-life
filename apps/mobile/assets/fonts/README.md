# Plus Jakarta Sans — Font Bundle

Drop the following `.ttf` files into this directory before building the app.

**Source**: https://fonts.google.com/specimen/Plus+Jakarta+Sans

Required files:
- `PlusJakartaSans-Regular.ttf` (weight 400)
- `PlusJakartaSans-Medium.ttf` (weight 500)
- `PlusJakartaSans-SemiBold.ttf` (weight 600)
- `PlusJakartaSans-Bold.ttf` (weight 700)
- `PlusJakartaSans-ExtraBold.ttf` (weight 800)

These files are NOT committed to the repository. Add them to `.gitignore` or commit
them directly if your team prefers bundled fonts in version control.

The font family name registered in `pubspec.yaml` is `PlusJakartaSans` — it must
match `BLType.family` in `lib/core/theme/bl_tokens.dart`.
